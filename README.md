# 🪙 Alke Wallet - Base de Datos Relacional

Este proyecto contiene el diseño e implementación de la base de datos relacional para la billetera virtual **Alke Wallet**, desarrollado dentro del curso Desarrollo de Aplicaciones Fullstack Python Trainee como parte del módulo de Fundamentos de Bases de Datos Relacionales.

## 🛠️ Tecnologías Utilizadas
* **Motor de Base de Datos:** PostgreSQL
* **Cliente SQL:** DBeaver
* **Control de Versiones:** Git / GitHub

## 📂 Estructura del Proyecto
* `alke_wallet_script.sql`: Script DDL y DML con la creación de tablas, inserción de datos y consultas.
* `alke_wallet_db_erd.png`: Modelo Entidad-Relación de la base de datos.
* `alke_wallet_db_doc.md`: Documento final con capturas y evidencias de ejecución.

## 🚀 Cómo ejecutar
1. Abre DBeaver y conéctate a tu servidor PostgreSQL local.
2. Abre el archivo `alke_wallet_db_script.sql`.
3. Ejecuta los bloques de código secuencialmente.
